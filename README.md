# Sistema Digital de Controle de Horas Extras – Projeto Integrador

Este projeto tem como objetivo desenvolver um protótipo em Linguagem C para simular o processo de registro, validação e aprovação de horas extras em empresas, substituindo o processo manual atual baseado em formulários e aprovações lentas. A automação busca reduzir erros, agilizar o fechamento de horas e melhorar a eficiência operacional.

O foco desta fase do projeto é a **lógica interna e o funcionamento do sistema**, com propósito acadêmico e de prototipagem.
